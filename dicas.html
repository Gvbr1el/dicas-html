Dicas de html para você que está começando

Quando se está começando a aprender a linguagem html, muitas vezes é difícil lembrar e saber algumas tags e funcionalidades.

Por isso iremos mostrar algumas tags e qual suas funcionalidades abaixo.

Primeiramente teremos a tag p, que serve para criarmos um parágrafo, e sua escrita se da da seguinte forma:

<p> aqui vai estar seu paragrafo </p>

Outra tag são as tags h, que servem para criar títulos e elas se dão desde h1 ate h6, sendo h1 o título mais importante da página e h6 o menos importante, ela se escrevem da seguinte forma:

<h1> aqui vai estar seu título </h1>

Por último temos a tag br, que serve para quebrar uma linha, ela pode ser escrita de duas formas pois é uma tag de autofechamento, ou seja só precisa colocar uma das duas maneiras,e elas são:

<br> ou </br>
